网上商城系统
===========

```开发环境```：IDEA，Tomcat，MySQL

```项目构建```：Maven

```软件环境```：SSH(SpringMVC，Spring，Hibernate) 

```项目描述```：项目旨在搭建一个线上购物商城系统，提供商城所具有的基本服务。基本可以进行用户注册与登陆、商品浏览、订单添加、购物车查看、虚拟货币结算、特定日优惠服务等一体化功能。
